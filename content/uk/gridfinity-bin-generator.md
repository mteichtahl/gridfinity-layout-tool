# Генератор Gridfinity bin

Параметричний генератор bin, що працює у вашому браузері. Задайте розміри, додайте відсіки чи вирізи, перевірте 3D-перегляд і завантажте файл STL, STEP або 3MF. Нічого встановлювати. Частина [пакету Gridfinity Generator](/uk/gridfinity-generator), куди також входять генератор основ і планувальник лейауту.

## Як це працює

1. **Задайте розмір.** Виберіть ширину, глибину та висоту в одиницях сітки. Потрібен розмір між стандартними? Режим напів-bin дає точність 0,5 одиниці.
2. **Додайте, що треба.** Відсіки, вирізи у стінках, скочи, етикетки — увімкніть і налаштуйте повзунками.
3. **Перевірте перегляд.** 3D-вид оновлюється миттєво, коли ви змінюєте налаштування. Обертайте і збільшуйте для кожного боку.
4. **Завантажте і друкуйте.** Експорт як STL для слайсера, STEP для редагування у CAD або 3MF для кольору і матеріалу.

> Уся обробка локальна. Ваші дизайни залишаються на пристрої, якщо ви ними не ділитесь.

## Функції

**Відсіки.** Розділіть bin на сітку секцій до 8×8. Перегородки генеруються автоматично. Знімні перегородки можна експортувати окремими файлами, щоб перенастроювати без передруку.

**Скочи і вирізи у стінках.** Скочи полегшують діставання дрібниць знизу. Вирізи у стінках — U-подібні насічки з будь-якого боку — дозволяють витягувати речі вбік.

**Етикетки.** Полиця на задній стінці тримає смужки етикеток. Виберіть кронштейн або суцільну опору і задайте ширину для кожного стовпця.

Для речей, що мають лишатися на місці, **вставки у дно** вирізають форму у підлозі — кола для батарейок, шестикутники для біт, прямокутники для компонентів. Налаштуйте глибину і поворот під те, що зберігаєте.

**Редактор вирізів** обробляє нестандартні форми. Перейдіть у суцільний режим, вирізайте зверху і використовуйте "перо" для довільних шляхів для ключів, пасатижів чи будь-якого профілю.

## Варіанти кріплення дна

Як bin з'єднується з вашою основою:

- **Стандарт** — типовий сокет Gridfinity
- **Магніт** — порожнини під магніти 6 мм × 2 мм
- **Гвинт** — отвори для різьбових вставок
- **Магніт і гвинт** — обидва способи для максимального утримання
- **Обважнене** — важче дно без магнітів
- **Плоске** — без сокета, для використання без основи

## Формати експорту

| Формат   | Коли використовувати                                                              |
| -------- | --------------------------------------------------------------------------------- |
| **STL**  | Прямо у ваш слайсер. Працює з PrusaSlicer, Cura, OrcaSlicer і всіма іншими.       |
| **STEP** | Відкрийте у Fusion 360, FreeCAD або SolidWorks для подальших правок перед друком. |
| **3MF**  | Містить колір і матеріал. Добре для багатоматеріальних принтерів.                 |

## Чому це, а не OpenSCAD?

Генератори Gridfinity на OpenSCAD працюють, але треба встановити ПЗ і редагувати код, щоб змінити один параметр. Цей працює у браузері з візуальним інтерфейсом. Результат бачите миттєво, без чекання на рендеринг.

Він також експортує STEP і 3MF — не лише STL — і працює на телефоні, якщо треба щось перевірити на ходу.
